
# Lojinha Online

## Descrição
Simulação de uma loja online em Java utilizando arquitetura monolítica.

## Funcionalidades
- Listagem de produtos
- Criação de pedidos
- Processamento de pagamento

## Singleton
O PagamentoService usa Singleton para garantir uma única instância.

## Execução
Compilar e executar a classe Main.
